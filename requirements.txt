# Python 3.10.x | Windows
#
# INSTALL IN THIS ORDER (mandatory — read below):
#
#   Step 1:  pip install -r requirements.txt
#   Step 2:  pip install crewai==0.86.0 --no-deps
#   Step 3:  pip install litellm opentelemetry-api opentelemetry-sdk
#
# WHY:
#   - CrewAI's version numbering skipped 0.12–0.85 on PyPI.
#     The real versions are 0.11.x then 0.86.x then 1.x.
#   - crewai>=0.86 is INDEPENDENT of LangChain (built from scratch).
#     Do NOT install langchain and crewai together via normal pip —
#     pip will try to satisfy crewai's chromadb>=1.0 requirement which
#     clashes with your 0.5.23 pin. --no-deps bypasses this.
#   - chromadb==0.5.23 is the last pure-Python backend.
#     0.6+ uses Rust bindings that crash on Windows Python 3.10.
#   - sentence-transformers 2.7.0 is the last version compatible with
#     chromadb 0.5.23's tokenizers<=0.20 ceiling.
#   - crewai-tools is NOT included — it pins exact crewai versions
#     and pulls embedchain which requires chromadb<0.5.0.
#     Our code uses crewai.tools.BaseTool directly (no crewai-tools needed).
#   - fastapi 0.111.0 + starlette 0.37.2 are pinned together because
#     fastapi 0.115 passes on_startup kwarg removed in starlette 0.41+.
# ─────────────────────────────────────────────────────────────
